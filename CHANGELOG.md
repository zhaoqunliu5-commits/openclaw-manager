# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-04-29

### Added

- **健康检查与自动恢复** — 定时检测服务存活状态，异常时自动重启并推送通知
- **数据导出** — 操作日志导出 CSV、监控数据导出 JSON、记忆数据导出 JSON
- **帮助文档面板** — 11 章节详细文档，含操作示例和提示，按 `?` 键或点击 📖 打开
- **中文搜索支持** — 技能搜索自动翻译中文关键词为英文（40+ 映射）
- **自动化预设模板** — 5 个常用模板一键创建（每日重启/服务停止通知/每小时检查等）
- **事件下拉选择** — 自动化规则创建时选择系统事件，无需手动输入
- **Agent/服务下拉选择** — 协作和自动化面板中选择 Agent 和服务，无需手动输入
- **规则可读化展示** — 自动化规则卡片显示可读的触发→动作描述
- **Toast 通知系统** — 全局错误提示，区分超时/认证失败/一般错误
- **新手引导** — 首次使用 5 步引导，引导用户查看帮助文档

### Fixed

- **监控数据空值兜底** — 安全数值转换，API 失败时显示友好提示而非空白
- **Agent 协作降级** — 三级数据源（CLI → 配置文件 → 目录扫描），空状态提示
- **记忆功能诊断** — 新增 WSL/路径/Agent 诊断，区分不同空状态原因
- **技能搜索降级** — CLI 失败时自动使用 GitHub Search API
- **ServiceCard PID 显示** — 修复 `||` 运算符对 PID 0 的误判，改用 `!= null`
- **ProcessCard 停止状态** — 服务停止时显示 `-` 占位符而非隐藏指标

### Changed

- **TypeScript 类型安全** — 消除 23 处 `any` 类型，新增 12 个接口定义
- **后端安全加固** — 添加 Helmet HTTP 头、Rate Limiting、输入校验
- **React 性能优化** — 关键组件使用 React.memo/useMemo/useCallback
- **缓存策略对齐** — 后端 DataCache (120s) 与前端 staleTime (30s) 对齐
- **API 错误处理** — 自动重试、错误分类（超时/认证/一般）、用户友好提示
- **自动化面板重构** — 完全重写 UI，增加模板/下拉/可读化/空状态引导
- **技能推荐搜索** — 新增搜索 Tab + 搜索输入框，支持中英文
- **README 更新** — 补充新功能、修正端口、添加快捷键表

## [1.0.0] - 2026-04-28

### Added

- 概览仪表盘 — 系统状态总览、关键指标统计
- 服务管理 — Gateway/Canvas 启停重启、状态监控
- Agent 管理 — Agent 列表、详情、模型配置
- 技能管理 — 技能列表、执行、搜索
- 配置管理 — 配置文件编辑、备份、恢复、差异对比
- 模型管理 — 模型提供商、Agent 模型分配
- 日志监控 — 系统日志、实时通知推送 (SSE)
- 工作区管理 — 工作区扫描、切换、备份恢复
- 监控面板 — 系统资源、Agent 活动、会话追踪
- 自动化引擎 — 定时/事件触发的规则执行
- 技能评估 — 技能使用统计、成功率分析
- GitHub 技能推荐 — 基于用户数据智能推荐
- 命令面板 — 快捷命令搜索与执行
- 记忆管理 — Agent 记忆条目、会话列表
- 多 Agent 协作 — 消息通信、路由绑定
